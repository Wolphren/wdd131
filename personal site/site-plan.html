Projects

I do not have the code for any of my projects at this moment so this will be a placeholder for now There will be defined sections for the different types of projects that are completed. I plan to add more projects as they become completed in the future in the following format:

  • Title
  • Description (possibility of image)

I am currently working on two different projects at the moment: One is a game where you change the player to the same color as the incoming obstacles which gets increasingly more fast paced the longer you play A Zork-esc game where it is mostly text-based and played in a terminal

Mindfulness Program

A terminal based program that has 3 different mindfulness exercises. A breathing exercise, a listening exercise, and a reflection exercise. For each you can do each for a time of your choosing before being escorted back to the main menu.

Journal Program

A terminal based Journal where you can write journal entries and are given a random prompt. You can display your current entries and then save them in a file. You can also load the file again later if you want to continue writing more journal entries.

Goal Setting Program

A terminal based program where you can create 3 different types of goals: a simple goal, a checklist goal, and an ongoing goal. You can then check of these goals for points and then display your total score.

Scripture Memorization Program

This is a terminal program with a set scripture. As you press the enter key you will slowly replace the words in the scripture with blank lines helping you memorize the scripture. The program ends when all the words have disappeared or you quit the program.
